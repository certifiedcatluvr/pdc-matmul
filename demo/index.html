01 Implementation Phases
PHASE 01
Sequential CPU Baseline
Standard triple nested loop in C. O(N³) complexity. Serves as the reference for all speedup calculations.
GCC · C
PHASE 02
CUDA Naive Kernel
One GPU thread per output element. 16×16 thread blocks. Global memory access — cache miss on column reads.
NVCC · CUDA
PHASE 03
CUDA Tiled Shared Memory
16×16 tiles loaded into shared memory. __syncthreads() synchronization. 10-100x lower memory latency than global memory.
NVCC · CUDA · Shared Mem
PHASE 04
OpenMP Parallelism
#pragma omp parallel for collapse(2) schedule(static). Combines both outer loops into a single parallel work pool.
GCC · OpenMP
PHASE 05
MPI Distributed
Row decomposition across 4 processes. MPI_Scatter distributes A rows. MPI_Bcast shares B. MPI_Gather collects results.
MPICH · MPI
PHASE 06
Docker Container
nvidia/cuda:11.8.0 base image. GCC + MPICH + NVCC + Python. Reproducible deployment across any compatible environment.
Docker · nvidia/cuda

05 Why CUDA Tiled Wins
🔵 CUDA Naive — Global Memory
Each thread reads directly from global memory on every iteration. Column-wise access to matrix B causes cache misses. Every element is loaded from global memory N times. Global memory latency: ~200-800 cycles.
// Each thread accesses global memory for (k = 0; k < N; k++) sum += A[row*N+k] * B[k*N+col]; // B[k*N+col] = cache miss every time
🟢 CUDA Tiled — Shared Memory
Tiles are loaded into shared memory once, then reused 16 times. Shared memory bandwidth is 10-100x faster than global. Each element loaded once per tile instead of N times. Shared memory latency: ~20-40 cycles.
__shared__ double tileA[16][16]; __shared__ double tileB[16][16]; // Load once, use 16 times __syncthreads(); // 10-100x faster access
06 Docker Containerization
Dockerfile
FROM nvidia/cuda:11.8.0-devel-ubuntu20.04 RUN apt-get update && apt-get install -y \ gcc mpich python3 python3-pip RUN pip3 install numpy matplotlib pandas WORKDIR /app COPY . . RUN gcc -O2 -fopenmp -o omp_matmul omp_matmul.c RUN mpicc -O2 -o mpi_matmul mpi_matmul.c RUN nvcc -O2 -o cuda_naive cuda_naive.cu RUN nvcc -O2 -o cuda_tiled cuda_tiled.cu CMD ["python3", "run_all.py"]
